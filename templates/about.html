About This Application

This web application is designed to help students of MAKAUT (Maulana Abul Kalam Azad University of Technology) convert their CGPA to percentage easily and accurately.

Features include:

  • Automatic CGPA to Percentage conversion based on official formula
  • Chart-based visual output
  • CSV data storage with timestamp
  • Secure admin panel to manage entries
  • Fully responsive design with Bootstrap
